# Counter App

A simple vanilla JavaScript counter application built to practice DOM manipulation and event listeners.

## Features
- Increase, decrease, and reset the counter value using buttons  
- Dynamically updates the counter color based on its value  
  - Green for positive numbers  
  - Red for negative numbers  
  - Neutral color when zero  
- Clean and responsive UI with instant feedback
